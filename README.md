# AI Prompt Manager

> A local AI prompt manager and text shortcut library generated with Microsoft Copilot Cowork.

# 📖 Usage Guide

## Creating a Prompt

1. Click **➕ New**
2. Enter:
   - A title
   - Your prompt content
   - A folder (optional)
   - Favorite status (optional)
3. Click **Save** 【1-b87670】

## Searching Prompts

Use the search bar to find prompts by:

- Title
- Prompt content

Results are filtered in real time as you type. 【1-b87670】

## Organizing Your Prompt Library

### Folders

Create folders to organize prompts by topic, project, or use case.

Examples:

- Marketing
- Development
- Meetings
- AI Engineering
- Translation
- Copilot Demos

From the **Folders** menu, you can:

- Create folders
- Rename folders
- Delete folders

Deleting a folder does not delete the prompts stored inside it. 【1-b87670】

### Favorites

Mark important prompts as favorites using the ⭐ icon.

Access all your favorite prompts instantly from the **Favorites** tab. 【1-b87670】

### Drag & Drop

Reorder prompts using drag-and-drop to create your own custom organization system. 【1-b87670】

## Using a Prompt

### Copy to Clipboard

Click **📋 Copy** to instantly copy a prompt to your clipboard. 【1-b87670】

### Open in Your Favorite AI Assistant

Select **↗️ Open In** and choose:

- Microsoft Copilot
- ChatGPT
- Google Gemini
- Any custom AI service

When launched:

1. The prompt is copied to the clipboard
2. The selected AI assistant opens automatically 【1-b87670】

## Configuring AI Services

Open **⚙️ Settings** to customize launch URLs.

Supported placeholder:

```text
{prompt}
```

Example:

```text
https://my-ai-service.com/?q={prompt}
```

This placeholder is automatically replaced with the prompt content before opening the service. 【1-b87670】

## Tracking Usage

Each time a prompt is copied or launched, its usage counter is updated.

This helps identify your most valuable and frequently used prompts. 【1-b87670】

## Backup and Restore

### Export

```text
Settings → Export
```

Downloads a JSON backup containing:

- Prompts
- Folders
- Preferences
- Usage statistics 【1-b87670】

### Import

```text
Settings → Import
```

Restore a previously exported library from a JSON file. 【1-b87670】

## Appearance

### Themes

Switch between:

- 🌙 Dark Mode
- ☀️ Light Mode

using the toolbar button. 【1-b87670】

### Views

Choose your preferred layout:

- **Grid View** for visual navigation
- **List View** for compact management of large libraries 【1-b87670】
